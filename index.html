CORSIA LCA Values

Feedstock CI values from ICAO CORSIA Default Lifecycle Values (LHV basis, gCO₂e/MJ). Actual values from supplier PoS documentation may differ.

Blend Limit

ASTM D7566 permits up to 50% SAF. WTW scope covers well-to-wake emissions. Non-CO₂ effects (NOx, contrails) are excluded per IATA scope.

Cost Context

SAF currently costs 2–5× conventional jet fuel. US IRA blending credits and EU SAF mandates (2025+) are improving economics. Prices vary by feedstock and region.
